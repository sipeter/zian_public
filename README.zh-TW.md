# Zian - 離線語音生產力工具

> **官方網站**：[ziansoft.com](https://ziansoft.com)  
> **你的資料，永遠屬於你的裝置。**  
> 專注於隱私安全、零延遲的本地化語音輸入解決方案。

## 📖 簡介
Zian 是一款打破雲端依賴的語音生產力工具。不同於傳統方案，Zian 直接利用您裝置的本地算力運行深度最佳化的 AI 模型。

無論您是在撰寫文件、即時聊天，還是整理會議錄音，Zian 都能提供毫秒級的響應速度和絕對的資料安全感。

## 🛡️ 隱私至上 (Privacy First)
我們深知資料的價值。Zian 嚴格遵循以下原則：
*   **完全離線**：全鏈路不依賴網路，拒絕上傳語音資料。
*   **零畫像**：不收集內容，不追蹤行為。
*   **透明可控**：模型、按鍵、行為邏輯完全由您掌控。

## ✨ 核心功能
*   **⚡ 極速輸入**：預設按住 `CapsLock` 說話，放開即上屏，擁有極佳的跟手感。
*   **💻 雙模式相容**：支援「剪貼簿貼上」與「模擬鍵盤輸入」，完美適配各類編輯器與遊戲。
*   **📂 Zian Scribe**：內建長音訊轉寫工具，支援批次處理與校對。
*   **📱 Zian Link**：區域網路內將手機化身為電腦的無線麥克風，擺脫硬體束縛。
*   **🧠 AI 智慧潤飾**：自動去除口語廢話，最佳化標點與段落。

## 💻 系統需求
在安裝前，請確保您的裝置滿足以下條件：
*   **系統**：Windows 10 / 11 (64位元)
*   **環境**：.NET Framework 4.7.2 或更高
*   **硬體**：建議雙核心 CPU + 8GB 記憶體（支援 GPU 加速）
*   **空間**：需預留足夠空間存放本地模型檔案

## 🚀 快速開始
1.  **解壓縮**：將壓縮包解壓至非中文路徑（推薦）。
2.  **執行**：雙擊 `Zian.exe`。
3.  **確認**：看到螢幕右下角出現托盤圖示，即表示準備就緒。
4.  **開始使用**：在任意輸入框，按住 **CapsLock** 說話，放開即可上屏。

> ⚠️ **注意**：首次運行若被 Windows SmartScreen 攔截，請選擇「仍要執行」。這是由於軟體包含鍵盤掛鉤與本地推理引擎，屬於正常現象。

## ⚙️ 常見問題 (FAQ)
*   **無法辨識？**
    請檢查 `core` 目錄下的模型檔案是否完整，且路徑中不要包含特殊字元。
*   **無法在遊戲中輸入？**
    請在設定中將輸入模式切換為「模擬鍵盤輸入」。
*   **詳細幫助文件**
    更多進階設定與使用技巧，請查閱軟體目錄下的 `Help` 資料夾。
