# Zian - 오프라인 음성 생산성 도구

> **귀하의 데이터는 영원히 귀하의 장치에.**  
> 개인정보 보호와 제로 레이턴시에 초점을 맞춘 로컬 음성 입력 솔루션.

## 📖 소개
Zian([ziansoft.com](https://ziansoft.com))은 클라우드 의존성을 타파한 음성 생산성 도구입니다. 기존 솔루션과 달리 Zian은 장치의 로컬 연산 능력을 직접 활용하여 최적화된 AI 모델을 실행합니다.

문서 작성, 즉각적인 채팅, 회의 녹음 정리 등 어떤 상황에서도 Zian은 밀리초 단위의 응답 속도와 절대적인 데이터 보안감을 제공합니다.

## 🛡️ 프라이버시 최우선 (Privacy First)
우리는 데이터의 가치를 깊이 이해하고 있습니다. Zian은 다음 원칙을 엄격히 준수합니다:
*   **완전 오프라인**: 전 과정이 네트워크에 의존하지 않으며 음성 데이터 업로드를 거부합니다.
*   **제로 프로파일링**: 콘텐츠를 수집하지 않으며 행동을 추적하지 않습니다.
*   **투명성과 통제권**: 모델, 키, 동작 로직은 모두 귀하가 통제할 수 있습니다.

## ✨ 핵심 기능
*   **⚡ 빠른 입력**: 기본적으로 `CapsLock`을 누른 채 말하고, 놓으면 입력됩니다. 뛰어난 조작감을 제공합니다.
*   **💻 듀얼 모드 호환**: "클립보드 붙여넣기"와 "키보드 입력 시뮬레이션"을 지원하여 각종 에디터 및 게임과 완벽하게 호환됩니다.
*   **📂 Zian Scribe**: 긴 오디오 파일 전사 도구를 내장하여 일괄 처리 및 교정을 지원합니다.
*   **📱 Zian Link**: LAN 내에서 휴대폰을 PC의 무선 마이크로 활용하여 하드웨어 제약에서 벗어납니다.
*   **🧠 AI 스마트 윤문**: 구어체 군더더기를 자동으로 제거하고 구두점과 단락을 최적화합니다.

## 💻 시스템 요구 사항
설치 전 장치가 다음 조건을 충족하는지 확인하십시오:
*   **시스템**: Windows 10 / 11 (64비트)
*   **환경**: .NET Framework 4.7.2 이상
*   **하드웨어**: 듀얼 코어 CPU + 8GB RAM 권장 (GPU 가속 지원)
*   **공간**: 로컬 모델 파일을 저장할 충분한 공간 확보

## 🚀 빠른 시작
1.  **압축 해제**: 압축 파일을 특수 문자가 없는 경로(권장)에 풉니다.
2.  **실행**: `Zian.exe`를 두 번 클릭합니다.
3.  **확인**: 화면 오른쪽 하단에 트레이 아이콘이 나타나면 준비 완료입니다.
4.  **사용 시작**: 임의의 입력란에서 **CapsLock**을 누른 채 말하고, 놓으면 입력됩니다.

> ⚠️ **주의**: 처음 실행 시 Windows SmartScreen에 의해 차단될 경우 "실행"을 선택하십시오. 이는 소프트웨어에 키보드 후킹 및 로컬 추론 엔진이 포함되어 있어 발생하는 정상적인 현상입니다.

## ⚙️ 자주 묻는 질문 (FAQ)
*   **인식 실패?**
    `core` 디렉터리 내의 모델 파일이 온전한지, 경로에 특수 문자가 포함되지 않았는지 확인하십시오.
*   **게임에서 입력 불가?**
    설정에서 입력 모드를 "키보드 입력 시뮬레이션"으로 전환하십시오.
*   **상세 도움말**
    더 많은 고급 설정과 사용 팁은 소프트웨어 디렉터리 내의 `Help` 폴더를 참조하십시오.

## 📄 라이선스 및 활성화
Zian은 상용 소프트웨어입니다.
*   **활성화**: "설정 -> 라이선스 관리"로 이동하여 일련번호를 입력하십시오.
*   **오픈 소스**: 이 프로젝트는 일부 오픈 소스 구성 요소를 사용합니다. 자세한 내용은 `THIRD-PARTY-NOTICES.txt`를 참조하십시오.
