# Zian - オフライン音声生産性ツール

> **あなたのデータは、永遠にあなたのデバイスに。**  
> プライバシー保護とゼロレイテンシーに特化した、ローカル音声入力ソリューション。

## 📖 はじめに
Zian ([ziansoft.com](https://ziansoft.com)) はクラウド依存を打破する音声生産性ツールです。従来のソリューションとは異なり、Zian はデバイスのローカル演算能力を直接利用して、高度に最適化された AI モデルを実行します。

ドキュメント作成、チャット、会議録音の整理など、どのような場面でも、Zian はミリ秒単位の応答速度と絶対的なデータ安心感を提供します。

## 🛡️ プライバシー・ファースト (Privacy First)
私たちはデータの価値を深く理解しています。Zian は以下の原則を厳格に遵守します：
*   **完全オフライン**：全プロセスがネットワークに依存せず、音声データのアップロードを拒否します。
*   **ゼロ・プロファイリング**：コンテンツを収集せず、行動を追跡しません。
*   **透明性と制御性**：モデル、キー、動作ロジックはすべてあなたが制御可能です。

## ✨ コア機能
*   **⚡ 高速入力**：デフォルトで `CapsLock` を押しながら話し、離すと入力されます。優れた操作感を実現。
*   **💻 デュアルモード互換**：「クリップボード貼り付け」と「キーボード入力シミュレーション」をサポートし、各種エディタやゲームに完全対応。
*   **📂 Zian Scribe**：長時間の音声ファイル文字起こしツールを内蔵し、一括処理と校正をサポート。
*   **📱 Zian Link**：LAN 内でスマートフォンを PC のワイヤレスマイクとして活用し、ハードウェアの制約から解放。
*   **🧠 AI スマート推敲**：話し言葉の無駄を自動的に取り除き、句読点と段落を最適化。

## 💻 システム要件
インストールの前に、デバイスが以下の条件を満たしていることを確認してください：
*   **OS**：Windows 10 / 11 (64bit)
*   **環境**：.NET Framework 4.7.2 以上
*   **ハードウェア**：デュアルコア CPU + 8GB メモリ推奨（GPU 加速対応）
*   **ストレージ**：ローカルモデルファイルの保存に十分な空き容量

## 🚀 クイックスタート
1.  **解凍**：圧縮ファイルを（推奨：日本語・特殊文字を含まない）パスに解凍します。
2.  **実行**：`Zian.exe` をダブルクリックします。
3.  **確認**：画面右下のトレイアイコンが表示されれば準備完了です。
4.  **使用開始**：任意の入力ボックスで、**CapsLock** を押しながら話し、離すと入力されます。

> ⚠️ **注意**：初回実行時に Windows SmartScreen にブロックされた場合は、「実行」を選択してください。これはソフトウェアにキーボードフックとローカル推論エンジンが含まれているためで、正常な現象です。

## ⚙️ よくある質問 (FAQ)
*   **認識できない？**
    `core` ディレクトリ内のモデルファイルが完全であるか、パスに特殊文字が含まれていないか確認してください。
*   **ゲームで入力できない？**
    設定で入力モードを「キーボード入力シミュレーション」に切り替えてください。
*   **詳細ヘルプ**
    その他の高度な設定や使用法については、ソフトウェアディレクトリ内の `Help` フォルダを参照してください。
