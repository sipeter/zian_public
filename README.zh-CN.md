# Zian - 离线语音生产力工具

> **你的数据，永远属于你的设备。**  
> 专注于隐私安全、零延迟的本地化语音输入解决方案。

## 📖 简介
Zian ([ziansoft.com](https://ziansoft.com)) 是一款打破云端依赖的语音生产力工具。不同于传统方案，Zian 直接利用您设备的本地算力运行深度优化的 AI 模型。

无论您是在撰写文档、即时聊天，还是整理会议录音，Zian 都能提供毫秒级的响应速度和绝对的数据安全感。

## 🛡️ 隐私至上 (Privacy First)
我们深知数据的价值。Zian 严格遵循以下原则：
*   **完全离线**：全链路不依赖网络，拒绝上传语音数据。
*   **零画像**：不收集内容，不追踪行为。
*   **透明可控**：模型、按键、行为逻辑完全由您掌控。

## ✨ 核心功能
*   **⚡ 极速输入**：默认按住 `CapsLock` 说话，松开即上屏，拥有极佳的跟手感。
*   **💻 双模式兼容**：支持“剪贴板粘贴”与“模拟键盘输入”，完美适配各类编辑器与游戏。
*   **📂 Zian Scribe**：内置长音频转写工具，支持批量处理与校对。
*   **📱 Zian Link**：局域网内将手机化身为电脑的无线麦克风，摆脱硬件束缚。
*   **🧠 AI 智能润色**：自动去除口语废话，优化标点与段落。

## 💻 系统需求
在安装前，请确保您的设备满足以下条件：
*   **系统**：Windows 10 / 11 (64位)
*   **环境**：.NET Framework 4.7.2 或更高
*   **硬件**：建议双核 CPU + 8GB 内存（支持 GPU 加速）
*   **空间**：需预留足够空间存放本地模型文件

## 🚀 快速开始
1.  **解压**：将压缩包解压至非中文路径（推荐）。
2.  **运行**：双击 `Zian.exe`。
3.  **确认**：看到屏幕右下角出现托盘图标，即表示准备就绪。
4.  **开始使用**：在任意输入框，按住 **CapsLock** 说话，松开即可上屏。

> ⚠️ **注意**：首次运行若被 Windows SmartScreen 拦截，请选择“仍要运行”。这是由于软件包含键盘钩子与本地推理引擎，属于正常现象。

## ⚙️ 常见问题 (FAQ)
*   **无法识别？**
    请检查 `core` 目录下的模型文件是否完整，且路径中不要包含特殊字符。
*   **无法在游戏中输入？**
    请在设置中将输入模式切换为“模拟键盘输入”。
*   **详细帮助文档**
    更多高级设置与使用技巧，请查阅软件目录下的 `Help` 文件夹。
